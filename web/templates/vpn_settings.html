Configuration Overview

This page allows you to configure the OpenVPN client configuration file (.ovpn) that users can download.

Required Fields
  • Server Address - Your VPN server hostname or IP
  • Port - VPN server port (default: 1194)
  • Protocol - UDP or TCP
  • CA Certificate - Certificate Authority certificate
  • Template - OpenVPN configuration template
Optional Fields
  • TLS Key - Additional TLS authentication (ta.key)
Users authenticate using their OpenVPN Manager credentials (auth-user-pass).
